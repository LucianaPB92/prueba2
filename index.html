Git y Github

Git

  • Es un software
  • Es mantenido por linux
  • Instalado localmente en el sistema
  • Es de codigo abierto
  • No tiene gestion de usuarios
  • se basa en un sistema de comandos
  • Control de versiones

Github

  • Es un servicio
  • Es mantenido por Mycrosoft
  • Es un repositorio remoto
  • Es gratuito y pago
  • Tiene incorporado la gestion de usuarios
  • Almacena versiones de repositorios de Git
